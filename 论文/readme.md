

# iSpace



## 提示词

这个项目是我的本科毕业设计，名称叫做 “iSpace 云盘系统”，你先熟悉整个系统的代码，了解功能和设计。

然后我需要你根据这个系统帮我写一篇高质量的毕业论文。

1. 你先参阅 @论文/自考毕业论文写作内容指南.md 中的内容，熟悉写作指南。
2. 然后参阅 @论文/我的文章 目录下的所有文章，这是我平时的写作风格。
    
    > 注意，让我参阅我的文章，不是为了让你借鉴内容，也并不是要你按照我的风格来输出内容，只是可以作为一个参考，目的是让你输出比较像 “真人” 的内容。

要求：

1. 不能抄袭，不能瞎编。
2. 内容风格不能有太多 AI 味道，要能过 AIGC 检测。
3. 正文 8000 - 12000 字。
4. 输出到 @论文/论文.md 文档中。

我有技术能力，只是表达能力欠缺，这篇论文对我及其重要，中间可能要经过多次修改。
