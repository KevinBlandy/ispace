
# ISpace

## 功能

### Member

* 系统概览
  * 占用空间
* 云文件
  * 全部文件/上传/下载/远程下载
  * 最近(全部/视频/图片/文档/压缩包/音频) video/audio/image/application/zip
  * 视频(day/week/month/year)
  * 图片(day/week/month/year)
  * 文档(day/week/month/year)
  * 音频(day/week/month/year)
  * 我的分享
    * 访问次数 
    * 取消（删除）/禁用
* 个人设置
  * 账户设置

### Manager

## 资源管理

* 系统概览
  * 磁盘占用/用户数量/资源数量/
  * 一个月上传数量/大小
  * 最新动态：xx 上传了 xxx / xx 删除了 / xx 了创建了
* 文件管理
  * 文件列表（文件/大小/状态/持有人/创建时间/修改时间）
* 会员关了
  * 会员列表(头像/账户/创建时间/修改时间)
* 系统管理
  * 系统配置

### zip 压缩

* 在线把多个资源压缩到一个文件中。

## 回收站功能设计

* 所有被删除的元素，都在回收站的一级目录下，除了目录的子元素
